The following is a collection of functionality provided by our Minion-API. Use this documentation as a way to build tools to interact with your minions remotely.

/speedtest
GET
 /speedtest/{html|json|xml} Fetch the latest speedtest results.
Request: URL is all that is required.
Response:
  • HTTP 200 - Returns status in requested format as a response.
  • HTTP 404 - Returns empty response if speedtest is not found.
/publicip
